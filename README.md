# JS Camp #3 – Caesar/TDD

Task – implement Caesar cipher using TDD approach, adjust app.ts to make it work in the browser.

## Install

```
git clone https://github.com/levaleks/js-camp-3-caesar-tdd.git
cd js-camp-3-caesar-tdd
npm install
```

## Lint, Test & Start

```
npm run lint
npm run test
npm run serve
```
